# Fetch the list of countries

Fetches the list of countries (ISO 3166-1) recognized by the GLEIF API.

## Usage

``` r
lei_countries()
```

## Value

A [`data.frame()`](https://rdrr.io/r/base/data.frame.html) with columns:

- **code**: The country code

- **name**: The country name

## Examples

``` r
if (FALSE) { # \dontrun{
lei_countries()
} # }
```
